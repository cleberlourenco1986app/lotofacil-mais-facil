LOTOFACIL BINHO - BASE PARA APP DE CELULAR

Este pacote tem 2 partes:

1. backend/
Servidor Python Flask que consulta resultados, calcula frequencias e gera jogos.
Precisa ficar online para o app funcionar fora do seu computador.

2. mobile/
App Android com Capacitor, visual de aplicativo e arquivos para abrir no Android Studio.

ORDEM CORRETA:
1. Publique o backend online.
2. Copie o link publico do backend.
3. Cole o link em mobile/www/config.js.
4. Rode mobile/CRIAR_ANDROID_STUDIO.bat.
5. Teste no Android Studio.
6. Gere o .aab para publicar na Google Play.

Documentos uteis estao na pasta docs/.
